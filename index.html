OS / Systems
OS Process Manager

A process scheduler simulator in C covering core OS concepts — scheduling, IPC, interrupts, system calls, and deadlock — across 25 modular source files.

  • Implemented 6 scheduling algorithms: FCFS, SJF, SRTF, Priority (preemptive & non-preemptive), and Round Robin driven by a global timer tick
  • Built deadlock detection using a Wait-For Graph with cycle detection and recovery by terminating the lowest-priority victim process
  • Implemented two IPC mechanisms: per-process mailboxes (point-to-point) and a system-wide message queue (broadcast), plus simulated syscalls and interrupt queue
  • Exports simulation results as JSON — Gantt chart, process stats, and WFG — viewable in a browser-based dashboard

Networking
TCP/HTTP Server

A portable, multithreaded HTTP/1.1 server built in C++ with no dependencies — manual parsing, real-time chat via SSE, and file sharing over raw sockets.

  • Wrote an HTTP/1.1 parser from scratch: request line, header normalization to lowercase, body extraction, and keep-alive connection management per request
  • Implemented real-time broadcast messaging using Server-Sent Events (SSE) and file upload/download over GET/POST
  • Thread-per-client model with std::mutex-protected client list; cross-platform socket abstraction supporting both POSIX and Winsock

Embedded · HAL
Temperature Sensor — STM32 Nucleo F401RE

Bare-metal-style embedded firmware on the STM32 Nucleo F401RE reading a DS18B20 sensor over 1-Wire, displaying readings on an I2C LCD, and streaming live data over UART.

  • Implemented the 1-Wire protocol driver from scratch using TIM1 for microsecond-precision timing — no library, no HAL wrapper
  • Wrote an I2C LCD driver handling 4-bit mode initialization, cursor positioning, and backlight control
  • Dual-mode firmware switchable via a single #define: a hardware test suite and normal operation mode with configurable high/low temperature thresholds
  • UART printf debugging at 115200 baud with a non-blocking main loop and modular driver design
